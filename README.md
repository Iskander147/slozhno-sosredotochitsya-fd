Этот сайт рассказывает:
- почему нам сложно концентрироваться
- как работает мозг при многозадачности
- что влияет на внимание (гаджеты, еда, стресс)
- способы улучшить концентрацию

Также реализован переключатель темы оформления.

---

## Функционал

- Адаптивная вёрстка
- Переключение темы:
  - светлая
  - тёмная (неон)
  - авто (системная)
- Семантическая HTML-структура
- Подключение кастомных шрифтов
- Галерея изображений
- Интерактивные кнопки смены темы (через JS)

---

##  Технологии

Проект использует:
- HTML5
- CSS3
- JavaScript (vanilla)
- Flexbox / Grid
- CSS-переменные (variables.css)
- Темизация (dark/light themes)
